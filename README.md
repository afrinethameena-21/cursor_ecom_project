# Cursor E-commerce Project
This project is an end-to-end ecommerce data pipeline built using Python, SQLite, Pandas, Faker, and Matplotlib.  
It covers synthetic data generation, database creation, exploratory data analysis (EDA), analytics, and automated report generation.
